# installation  

- Run 
```
composer install
````
- Change .env to correct username and password for database

```
DB_USERNAME=safwat
DB_PASSWORD=123456
```


- Run 
```
php artisan migrate
````

- To run seed need to run

```
php artisan db:seed
```
